Connect your WordPress website to the Wovax CRM dashboard.

== Description ==

Looking to connect your WordPress website to [Wovax CRM](https://crm.wovax.com/ "Wovax CRM")?

This plugin requires an active subscription to [Wovax CRM](https://crm.wovax.com/ "Wovax CRM"). Use this plugin to generate and display contact forms on your WordPress website. When a website visitor submits the contact form the plugin can route the details to an email address and/or create and new contact profile in your Wovax CRM dashboard.


**Specs & Features**

* Display contact forms throughout your website via the shortcode [wovax-crm-contact-form].


**Coming Soon**

* Automatically add new contacts to your [Wovax CRM](https://crm.wovax.com/ "Wovax CRM") dashboard when website visitors submit the contact form.

* Gain insight into your website users favorite [Wovax IDX](https://wovax.com/products/idx/ "Wovax IDX") property listings and property search activity.

== Installation ==
1. Download and extract the Wovax CRM API plugin. You may also install the plugin directly by going to your WordPress Dashboard and then navigating to Plugins > Add New. Search for 'Wovax CRM API' under new plugins and click to install directly.
2. Upload the uncompressed folder to your `/wp-content/plugins/` directory.
3. Activate the Wovax CRM API plugin through the plugins menu in WordPress.

== Frequently Asked Questions ==
= Does this plugin require a subscription to Wovax CRM? =
This plugin can be used to display contact forms that submit to an email address without a subscription. If you'd like to use this plugin with Wovax CRM, then you need to sign up for a monthly or yearly plan for Wovax CRM.

= How do I display the contact form on a post or page? =
Simply copy the shortcode [wovax-crm-contact-form] and paste where you'd like to display the contact form.
